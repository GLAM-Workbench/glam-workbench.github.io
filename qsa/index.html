    Naturalisations, 1851-1904 add series information¶

    The Naturalisations, 1851 to 1904 index is available from the Queensland Government data portal. It's not clear, however, that the index collates name entries from a number of different series, with a separate row for each name reference. This means that there can be multiple rows referring to the naturalisation of a single individual. This is obviously important to keep in mind if you're trying to analyse aggregate data relating to naturalisations in Queensland. This notebook adds series information to the original index so that you can filter the data by series.
